# Cloud Based Scalable E-Commerce System with CI/CD on AWS

## Project Overview 
This project is a cloud-based scalable e-commerce application deployed on AWS using Docker and CI/CD automation.

## Technologies Used
- AWS EC2
- Amazon S3
- Docker
- Github Actions
- React.js
- Node.js
- Express.js

## Features
- Scalable cloud deployment
- Containerized application 
- Automated CI/CD pipleine
- Frontend hosting using S3
- Backend deployment on EC2

## CI/CD Workflow
Github Actions automatically:
- Builds the application
- Runs deployment steps
- Updates Docker containers on EC2

## AWS services used
- EC2, S3, IAM, Auto Scaling, Load Balancer

## Project Structure
frontend/
backend/
.github/workflows/

## How to run project
1. Clone repository
2. Install dependencies
3. Run docker containers
4. Deploy on AWS
